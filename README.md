## 🌐 About the Project

**GyaanSetu** is a next-generation EdTech platform built to solve India's most critical education crises: the **practical skills gap** and the **regional language barrier**. 

While theoretical knowledge is abundant, the ability to apply skills in industry-simulated environments is lacking. Furthermore, millions of talented students are excluded from quality education simply because it is delivered exclusively in English. GyaanSetu solves this by delivering high-quality technical education in **Hindi**, **Gujarati**, and **English**, coupled with a **browser-based coding environment**.

---

## 🏛️ The Dual-Pillar Solution

### 🧪 Pillar 1: Hands-On Practical Labs
A browser-based coding environment that lets students learn by doing — exactly as they would in a real job.
- **Monaco Editor:** Industry-standard code editing experience.
- **Simulated Execution:** Run code and see immediate outputs.
- **Auto-Evaluation:** Instant feedback with test case pass/fail metrics.

### 🌍 Pillar 2: Regional Language Delivery
Complete content localization that brings elite-quality education to students in their mother tongue.
- **Bilingual Interface:** Real-time switching between languages across the entire UI.
- **Localized Content:** Course instructions, lab descriptions, and feedback in Hindi and Gujarati.

## ✨ Key Features

### 🔐 Authentication & Onboarding
- **Multi-method Login:** Email/Password, Google, and GitHub OAuth support.
- **Language Preference:** Select your preferred learning language during onboarding.
- **Role-based Access:** Tailored experiences for Students and Instructors.

### 📊 Personalized Dashboard
- **Real-time Stats:** Track courses enrolled, labs completed, and XP earned.
- **Streak System:** Gamified daily login rewards to boost engagement.
- **Activity Heatmap:** Visual representation of learning consistency (GitHub-style).

### 📚 Course Learning Experience
- **HD Video Player:** Integrated with regional language subtitle support.
- **Curriculum Tracking:** Expandable modules with progress indicators.
- **Mark as Complete:** Systematic tracking of lesson completion.

### 🧪 Lab & Project Workspaces
- **3-Panel Lab View:** Instructions | Code Editor | Terminal Output.
- **Kanban Milestones:** Track project progress from "To Do" to "Done."
- **Multi-language Support:** Labs available in Python, JavaScript, Java, and C++.

### 🏆 Community & Gamification
- **Leaderboards:** Weekly, Monthly, and All-Time global rankings.
- **Achievement Badges:** 20+ unlockable badges for milestones and streaks.
- **Instructor Profiles:** Public pages for creators with ratings and reviews.

## 🛠️ Tech Stack

### Frontend Core
- **React 18 + Vite:** Fast, modern UI development.
- **Redux Toolkit:** Robust global state management (Auth, UI, User, Language).
- **React Router 6:** Complex client-side routing.

### UI & Styling
- **Tailwind CSS:** Utility-first styling for the "Radiant Scholar" design system.
- **Material UI (MUI):** Integrated for complex interactive components.
- **Framer Motion:** Premium micro-animations and page transitions.
- **Lucide React:** Accessible, consistent iconography.

### Infrastructure & Utilities
- **i18next:** Multi-language infrastructure.
- **React-Helmet-Async:** Systematic SEO and metadata management.
- **Axios:** Hardened API layer with global error interceptors.
- **Google Analytics (GA4):** Professional-grade event tracking.

## 🏁 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
```bash
# Clone the repository
git clone https://github.com/DhruvOzha85/GyaanSetu.git

# Navigate to project directory
cd GyaanSetu

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env and add your VITE_GA_ID
```

### Commands
```bash
# Start dev server
npm run dev

# Run linting
npm run lint

# Build for production
npm run build
```
